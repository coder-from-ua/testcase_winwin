# Local Reverse Proxy Demo (Docker Compose)

Tiny web service running behind an nginx reverse proxy with rate limiting and request ID handling.

---

## 1) Prerequisites

Required:
- Docker
- Docker Compose v3+

---

## 2) How to run service

```bash
make up

---

## 3) How to test service

```bash
make test

---

## 4) How to stop service

```bash
make down

---

## 4) How to log service

```bash
make logs
