# Local Reverse Proxy Demo (Docker Compose)

Tiny web service running behind an nginx reverse proxy with rate limiting and request ID handling.

---

## Prerequisites

**Required:**
- Docker
- Docker Compose v3+

---

## How to Run

Start the service:
```bash
make up
```

---

## How to Test

Run tests:
```bash
make test
```

---

## How to Stop

Stop the service:
```bash
make down
```

---

## How to View Logs

View service logs:
```bash
make logs
```
